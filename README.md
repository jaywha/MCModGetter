# MCModGetter
Basic mod manager for vanilla MC launcher.

## Latest Release
[v0.5 Executable](https://github.com/jaywha/MCModGetter/releases/download/v0.5/MCModGetter.exe)  
To install, unzip the above file and run the **setup.exe** file.
